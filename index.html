About This Dashboard

This project started as an entry for the Talent Intelligence Collective "One and Done" challenge with inDrive. In that contest, participants had one prompt attempt to generate a full location strategy report across LATAM, APAC, and Africa & Middle East. This entry won the contest.

Contest details are available here: The TIC One and Done Challenge.

  • How this app was built: The original prompt was authored for Lovable, which generated the initial `index.html` code for this dashboard.
  • What is static in this code: All city scores, salary benchmarks, strengths/challenges, methodology, and Q&A entries are preloaded static data points inside this HTML file.
  • What LLM calls are used for: Gemini is used only for interactive features: processing custom scenarios, answering chatbot questions using the report context, and generating city-specific recruitment battle cards.
  • What does not change automatically: Baseline report data does not auto-refresh from external data sources; it changes only through code edits or user-triggered AI transformations.
